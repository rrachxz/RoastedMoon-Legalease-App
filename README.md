# LegalEase
Bridging the Legal Literacy Gap. Legalease is an AI-powered app designed to protect elderly communities and vulnerable groups from digital contract scams and predatory terms.

> Built for **GDGoC KitaHack 2026** by Team RoastedMoon.

# Features

* **AI Document Analysis** - Instant legal document review with key points extraction
* **Conversational AI Chat** - Ask questions about your documents in natural language
* **Multi-Format Support** - PDF, DOC, and DOCX file analysis
* **Document History** - Cloud-synced storage of all analyzed documents
* **Jargon Decoder** - Quick legal term definitions and explanations
* **Scam Alerts** - Scanning for common scam patterns and red flags in documents
* **Legal News Feed** - Stay informed with latest legal tech articles
* **Real-time Sync** - Document history synchronized across devices

## Technologies

* **Flutter** - Cross-platform mobile framework
* **Dart** - Programming language
* **Google Gemini API** - AI document analysis and chat
* **Firebase Authentication** - User management and security
* **Cloud Firestore** - NoSQL database for real-time sync
* **NewsAPI** - Real-time scam trend updates
* **ML Kit** - Text recognition (planned)
